About the Project

PUK @ Time capsule is a blockchain-based application designed for secure and transparent message logging, styled with a crypto warrior samurai theme. It combines blockchain technology, a community discussion board, and a token-based economy to create an immersive experience for crypto enthusiasts.

PUK @ คัมภีร์สายกระบี่คริปโต เป็นแอปพลิเคชันที่ใช้เทคโนโลยีบล็อกเชนเพื่อบันทึกข้อความอย่างปลอดภัยและโปร่งใส ออกแบบในสไตล์นักรบซามูไรแห่งโลกคริปโต โดยผสานเทคโนโลยีบล็อกเชน กระดานสนทนาชุมชน และระบบเศรษฐกิจที่ใช้โทเค็น PUK เพื่อมอบประสบการณ์ที่ดื่มด่ำสำหรับผู้ที่ชื่นชอบคริปโต

Key Features / คุณสมบัติหลัก

  • Blockchain Message Logging: Store messages securely in the "Time Capsule" with immutable blockchain records.
    บันทึกข้อความบนบล็อกเชน: ฝากข้อความใน "Time Capsule" อย่างปลอดภัยด้วยบันทึกบล็อกเชนที่ไม่สามารถเปลี่ยนแปลงได้
  • PUK Token System: Earn PUK tokens by creating blocks and use them for trading or transfers.
    ระบบโทเค็น PUK: รับโทเค็น PUK ผ่านการสร้างบล็อกและใช้ในการซื้อขายหรือโอน
  • Dojo Board: A community discussion board with XSS protection for posts and comments.
    กระดาน Dojo: กระดานสนทนาชุมชนที่มีการป้องกัน XSS สำหรับกระทู้และความคิดเห็น
  • Admin Dashboard: Admins can manage posts, comments, and register new admins securely.
    แดชบอร์ด Admin: ผู้ดูแลระบบสามารถจัดการกระทู้ ความคิดเห็น และลงทะเบียน Admin ใหม่ได้อย่างปลอดภัย
  • Wallet Management: View balances and perform transactions with a user-specific crypto wallet.
    การจัดการกระเป๋าเงิน: ดูยอดคงเหลือและทำธุรกรรมด้วยกระเป๋าเงินคริปโตเฉพาะผู้ใช้
  • PUK Trading: Buy or sell PUK tokens with prices linked to Binance API.
    การซื้อขาย PUK: ซื้อหรือขายโทเค็น PUK โดยใช้ราคาที่เชื่อมโยงกับ Binance API

Technical Details / รายละเอียดทางเทคนิค

  • Backend: Flask (Python) with PostgreSQL for secure data storage.
    Backend: Flask (Python) พร้อม PostgreSQL สำหรับจัดเก็บข้อมูลอย่างปลอดภัย
  • Frontend: Bootstrap with neon-themed CSS and JavaScript animations (sword-slash effects, toast notifications).
    Frontend: Bootstrap พร้อม CSS ธีมนีออนและอนิเมชัน JavaScript (เอฟเฟกต์ดาบฟัน, การแจ้งเตือน toast)
  • Security: Environment variables (.env) for sensitive data, input sanitization with Bleach to prevent XSS.
    ความปลอดภัย: ตัวแปรสภาพแวดล้อม (.env) สำหรับข้อมูลที่ละเอียดอ่อน, การ sanitize อินพุตด้วย Bleach เพื่อป้องกัน XSS
